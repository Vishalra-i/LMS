LMS API Documentation

User Routes

POST /api/v1/users/register

Register a new user.

POST /api/v1/users/login

Log in a user.

GET /api/v1/users/profile

Get user profile data (requires authentication).

GET /api/v1/users/logout

Log out the current user.

POST /api/v1/users/forgot-password

Request a password reset.

POST /api/v1/users/reset/:resetToken

Reset the user's password using the reset token.

PUT /api/v1/users/update/:id

Update user profile information (requires authentication).

Course Routes

GET /api/v1/courses

Retrieve all courses.

POST /api/v1/courses

Create a new course (Admin only).

GET /api/v1/courses/:id

Get lectures for a specific course (requires subscription).

POST /api/v1/courses/:id

Add a lecture to a course (Admin only).

PUT /api/v1/courses/:id

Update a course's information (Admin only).

DELETE /api/v1/courses/:courseId/lectures/:lectureId

Remove a lecture from a course (Admin only).

Payment Routes

POST /api/v1/payments/subscribe

Subscribe to a course (requires authentication).

POST /api/v1/payments/verify

Verify subscription payment (requires authentication).

POST /api/v1/payments/unsubscribe

Unsubscribe from a course (requires authentication).

GET /api/v1/payments/razorpay-key

Get the Razorpay API key for payments (requires authentication).

GET /api/v1/payments

Retrieve all payments (Admin only).

Miscellaneous Routes

POST /api/v1/contact

Send a contact message.

GET /api/v1/admin/stats/users

Get user statistics (Admin only).
